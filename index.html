Time to Analyze Weather!


The purpose of this project is create a visualization dashboard website using visualizations to analyze how weather changes depending on your proximity to the equator. To accomplish this analysis, I first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After assembling the dataset, I used Matplotlib to plot various aspects of the weather vs. latitude. Elemental factors I took into considerate are:

Temperature, Cloudiness, Wind Speed, and Humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.


To see a visualization, select an option on the "Plots" dropdown menu.
